# Gaming Website Page

This repository contains the code for a simple gaming website page. The page is designed to showcase a gaming controller product and includes a navigation bar, product details, and social media links. The code is written in HTML and includes some basic CSS styling.

## Table of Contents

- [Getting Started](#getting-started)
- [Usage](#usage)
- [File Structure](#file-structure)
- [Contributing](#contributing)
- [License](#license)

## Getting Started

To get started with this code, simply clone the repository to your local machine.

```bash
git clone <repository-url>
```

Open the HTML file (`index.html`) in a web browser to view the gaming website page.

## Usage

The website page is a simple showcase of a gaming controller product. It includes a navigation bar with links to different product categories and product details with an option to buy. The social media links are also provided at the bottom of the page.

## File Structure

- **index.html**: The main HTML file containing the structure of the gaming website page.
- **style.css**: The CSS file for styling the HTML elements.
- **images/**: A directory containing images used in the website, including the logo, menu icon, product image, and social media icons.
